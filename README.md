# APS Developer - Landing Page

Sitio web corporativo premium para APS Developer con React, Next.js, TailwindCSS y Framer Motion.

## 🚀 Características

- ✅ Hero section responsive
- ✅ Calculadora interactiva de presupuesto
- ✅ Chatbot conversacional con IA
- ✅ Integración WhatsApp (+54 9 3755 528367)
- ✅ Portfolio con 6 proyectos ficticios
- ✅ Testimonios y proceso de trabajo
- ✅ Diseño oscuro premium con naranja
- ✅ Animaciones fluidas con Framer Motion

## 🛠 Tecnologías

- **Framework**: Next.js 16.2.9
- **React**: 18.3.1
- **Estilos**: TailwindCSS 3.4.4
- **Animaciones**: Framer Motion 11.3.7
- **Iconos**: Lucide React
- **Hosting**: Vercel

## 📋 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/tuusuario/aps-developer-landing.git
cd aps-developer-landing

# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev

# Construir para producción
npm run build

# Iniciar producción
npm start
```
